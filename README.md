# HTML & CSS Project

----- Features --------

a. Semantic HTML5 structure
b. Responsive layout using Flexbox/Grid
c. Clean and minimal CSS styling
d. Cross-browser compatibility

------ Technologies Used -----

1. HTML5
2. CSS3 (Flexbox, Grid, Media Queries)
